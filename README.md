<h1 align="center">Bootstrap Modern Unix</h1>


<p align="center">
  Bootstraps and configures the environment for modern cli development.
  It aims to be a curated list of utilities that aim for Safety, Speed, and Ease of Use.
</p>

  </h1>
  <p align="center">
